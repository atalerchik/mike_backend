# Mike Backend

## Installation

Clone the repository:

```bash
git clone git@github.com:Talerchik-Aleksey/mike_backend.git
```

## Usage

```bash
cd mike_backend
npm install
```

Run migration for database:

```bash
npm run migration:run
```

Run tests:

```bash
npm run test
```

Run server:

```bash
npm run dev
```
